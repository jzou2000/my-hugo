## if-else

Cpp, Java, javascript
```cpp
if (condition) {
    ;
}
else if (condition) {
    ;
}
else {
    ;
}
```

Python
```python
if condition:
    pass
elif condition
    pass
else:
    pass
```

Perl
```perl
expr if (condition);
expr unless (condition);
```



## for

```cpp
for (int i = 0; i < 10; ++i) {
    action();
}
```

```python
for i in range(10):
    action()
```

```js
for (prop in object) {
    action()
}
for (element of array) {
    action()
}

```
